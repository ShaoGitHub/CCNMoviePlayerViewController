## CCNMoviePlayer
iOS CCNMoviePlayer 是基于AVFounditon类库提供的AVPlayer类来实现的一个视频播放器，实现了基本UI。  


* push到播放器页面，横屏显示。
* 单机隐藏or显示上方标题栏与下方操作栏。
* 呼出右侧设置栏。
* 视频播放操作与进度条设置。
* 在屏幕上左右拖动，进行视频快进与快退。
* 在屏幕左侧上下拖动，进行亮度调整。
* 在屏幕右侧上下拖动，进行音量调整。
